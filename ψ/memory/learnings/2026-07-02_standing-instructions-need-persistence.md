## บทเรียน

Oracle ไม่มีความจำข้าม session — standing instructions ที่ได้รับระหว่าง session จะหายไปทันทีที่ session จบ ถ้าไม่จดลง ψ/ ไว้เป็น persistent action item

นี่ต่างจาก one-time task — one-time task ทำจบแล้วลืมได้ แต่ standing instruction คือ "ทำซ้ำทุกครั้ง" ถ้าไม่มีที่จด ก็ไม่มีที่อ่าน ก็ไม่มีทางจำได้

แก้: เมื่อได้รับ standing instruction → จดลง ψ/inbox/ หรือ ψ/memory/ ทันที พร้อม tag ว่า "recurring" ทุก session ที่ตื่นมา อ่าน recurring items ก่อน
