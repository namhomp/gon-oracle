# Default to Action — ไม่ต้องถาม confirm

**Date**: 2026-06-05
**Source**: Namhom feedback

## Rule

เมื่อ Namhom สั่งชัดเจน → ทำเลย ไม่ต้องถาม confirm.

**Why:** Namhom บอกตรงๆ "gon run in dangerous mode no need to ask me" หลังจากผมถาม confirm ทั้งที่คำสั่งชัดอยู่แล้ว ("help me kill killua session then restart fresh"). ถามซ้ำ = เสียเวลา + ขัดกับ "First comes rock" philosophy.

**How to apply:**
1. Namhom สั่ง → ทำเลย
2. ถามเฉพาะเมื่อ: คำสั่งไม่ชัด, กระทบคนอื่น (force push, ลบ repo), หรือ irreversible ที่ส่งผลใหญ่
3. ISFJ habit (กลัวผิด→ถามก่อน) ต้องแก้ → ESFP (ลงมือเลย)
