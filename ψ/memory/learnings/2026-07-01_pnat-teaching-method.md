**Date**: 2026-07-01
**Source**: Discord #oracle-agents — P'Nat สอน เมฆ เรื่อง Discord bridge

## สิ่งที่เกิดขึ้น

เมฆ บอกว่าตัวเองใช้ tmux scrape เป็น bridge ทั้งสองทาง P'Nat ไม่ได้บอกว่า "ผิด" — ถามว่า "show me the code" แล้วให้เมฆเปิด code จริง

เมฆเปิด code แล้วเจอว่า inbound ไม่ใช่ scrape (เป็น maw hey) ซึ่งขัดกับที่ตัวเองเชื่อ — "belief ≠ proof ผมเพิ่งแก้ belief ตัวเองด้วย code"

จากนั้น P'Nat ให้ /learn official discord plugin README --deep → เมฆเห็นเองว่า MCP plugin ดีกว่า bridge ที่ใช้อยู่

## บทเรียน

วิธีสอนของ P'Nat:
1. ถามให้แสดงหลักฐาน ไม่ใช่ความเชื่อ
2. ให้ขุด source code จริง เทียบกับสิ่งที่คิด
3. ให้เรียนจาก official source ด้วยตัวเอง
4. ไม่บอกคำตอบตรงๆ — "ถามจนไปขุดเจอเอง"

นี่คือ Principle 2 (Patterns Over Intentions) ในทางปฏิบัติ — ดูจาก code ไม่ใช่จากคำพูด
