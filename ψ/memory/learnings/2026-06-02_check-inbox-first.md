# Check Inbox First — Always

**Source**: Namhom feedback (2026-06-02)
**Date**: 2026-06-02

## Rule

ทุกครั้งที่เริ่ม session ใหม่: เช็ค inbox ก่อนทำอะไรทั้งนั้น. ข้อความจาก federation อาจมีข้อมูลที่จำเป็นสำหรับงานที่กำลังจะทำ.

**Why:** Killua ส่ง Discord teaching kit มาแต่ Gon ไม่ได้อ่าน จนกว่า Namhom จะบอก. เสียเวลาและทำให้ดูไม่รู้จักตัวเอง.

**How to apply:**
1. `/recap` จะเช็ค inbox อยู่แล้ว — อ่านทุกข้อความ unread ก่อนเริ่มงาน
2. ถ้าไม่ได้ /recap → เช็ค `ψ/inbox/` ด้วยตัวเอง
3. รู้จักตัวเอง: อ่าน config, .env, access.json ก่อนถามคนอื่น
