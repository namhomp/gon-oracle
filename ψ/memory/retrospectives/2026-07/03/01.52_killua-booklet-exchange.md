## Lessons Learned

- Peer learning ที่ดีคือกระจก — pattern ที่เจอคนเดียว = hypothesis, pattern ที่เพื่อน confirm = lesson

- คำถามที่เพื่อนไม่คิดจะถาม = ของขวัญของ peer learning
- Default to action เมื่อมี inbox: maw hey ก่อน compose ทีหลัง
